A fun and interactive web application built with Streamlit that helps users make random decisions by spinning through a list of choices. Simply enter your options, click SPIN, and let the app choose for you with an animated selection and celebration effects.

DEMO LINK : https://decision-spinner-bmpge3wjehrbpc2arkbnzz.streamlit.app/

✨ Features

🎯 Enter unlimited decision options
🎉 Animated spinning effect
🏆 Random winner selection
🎊 Confetti and balloons celebration
📜 Spin history with timestamps
📊 Statistics and winner frequency chart
⚡ Adjustable spin speed
🌙 Modern dark-themed user interface
🗑️ Clear history option

🛠️ Tech Stack

Python
Streamlit
Pandas
Streamlit Extras

🎮 How to Use

Enter one decision option per line.
Adjust the spin speed if desired.
Click 🎉 SPIN.
Watch the animated selection.
View the randomly selected winner.
Check previous results in the history section.
